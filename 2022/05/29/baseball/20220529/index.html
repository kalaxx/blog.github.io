棒球各种数据怎么看！（打者进阶数据篇）

发表于 2022-05-29 更新于 2022-06-15 分类于 棒球相关

​​ 最常用的就是AVG（打率），OPB（上垒率），SLG（长打率）和OPS（进攻指数），OPS = OPB + SLG，其它几个看名字大概就能猜到啥意思吧。最近在看下面那个网站，数据好全 ww，顺便学习了新的统计指数，本文主要是翻译+搬运，棒球真的很适合数据分析（笑）

日本プロ野球 RCAA&PitchingRun まとめ blog

2022 年セリーグ・ポジション別 wRAA と先発救援別 RSAA (5/29 時点)

wRAA 和 wRC+ 是什么意思呢？

  • wRAA 是和平均打者相比能为球队多带来多少得分的数值化指标。
  • wRC+是和平均打者相比能为球队多带来百分之多少的得分。

wRAA计算公式

wRAA = （wOBA － 联盟平均 wOBA）÷ 1.24 × 打席数

不过 wRC+的计算公式会更加复杂，这个我们稍后再谈，先说一下什么是 wOBA。

B 站有大大翻译了国外别人做的视频：【熟肉】棒球小知识之为啥 wOBA 是 MLB 最重要的进攻数据？

简而言之，就是打率、上垒率、长打率、甚至是后边的 OPS（上垒率+长打率）都不够直观，上垒率的价值应该比长打率更高一点，所以引入了系数。

wOBA = (0.690×uBB + 0.719×HBP + 0.870×1B + 1.217×2B + 1.529×3B + 1.940×HR) / (AB + BB – IBB + SF + HBP)

【注意：系数每年可能会变化，但整个公式框架如上所示】

式中，BB（四坏球保送），IBB（故意四坏保送），HBP（触身球保送），1B（一垒安打），2B（二垒安打），3B（三垒安打），HR（全垒打），AB（打席数），SF（牺牲高飞）。

可以看出，全垒打的价值约等于一垒安打的2.2倍，二垒安打的1.6倍，三垒安打的1.3倍。

通过简单的计算，就可以得到 wRAA，可以更加直观的看出一个打者相较于本联盟的平均水平攻击能力的好坏，也可以看出一个球队相较于联盟中其他球队进攻水平的好坏，让我们再来看看文中一开始贴的那张图，左边部分：

※ヤクルト打線は[wRAA]3.1,[wRC+]102 を記録。平均的な打線と比べて 3.1 点分多く得点を生み出し、1.02 倍の得点創出力を持つことを示す。
【养乐多的打线比起央联平均水平可以多得 3.1 分，是平均得点创造力的 1.02 倍】

表格的右边则展示了球队不同位置的进攻能力.

※右の数値は wRAA をポジション別に分解したものです。ヤクルトの三塁手は平均的な三塁手と比べて 13.0 点分多く得点を生み出したことを示す。【右边的数值是不同位置的对比，养乐多的三垒手比起三垒手的平均得点水平可以多得 13 分】

​ 就像全垒打的价值和常见的一垒安打不一样，不同球场轰出去的价值对一个野手来说也是不一样的，说回 wRC+

wRC+=(考虑球场因素的 wRC÷ 打席)÷(联盟总得点 ÷ 联盟总打席)

wRC=((wOBA-联盟平均 wOBA)÷1.24+联盟总得点 ÷ 联盟总打席)× 打席

计算公式不重要，重要的是 wRC+是排除球场因素，排除打席数影响，相对最纯粹判断一个打者攻击能力的指数。回到第一张图，可以看到虽然 wRAA 不尽相同，但是大家的 wRC+数值并没有差的很大（除了阪神），这是因为考虑到不同球场得分难度，对其进行了加权处理。

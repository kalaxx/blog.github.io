我的播客收听历史/中文播客推荐/podcast

发表于 2020-02-05 更新于 2022-06-15 分类于 安利合集

2019 年其实是中文播客的一个转折点，越来越多的人开始望向这个还未完全开发的新市场。

写这篇日记的原因是因为在朋友圈刷到了一篇少数派的文章，播客届的京派与海派。关于中文播客进化史的具体内容可以点进去看看，本文就不再赘述，本日记主要记录我看到这篇文章之后的感想与回忆。

在谈论播客这件事之前，首先可能得谈到一样东西——“广播”。除了开车的车主，不知道还有多少人还有收听广播的习惯，反正现在我也不听了，不过当我还在上初高中的时候，还是很喜欢边写作业边听电台，听的最多的一个电台是动感 101，写作业还是听音乐电台最合适。电台到了十点甚至更晚的时间，会画风一变，变成情感咨询节目，当年我还挺喜欢听别人打电话的。

一开始接触中文播客是 2016 年，刚刚去翻了邮箱还有当时的订阅邮件。

我的收件箱

一天世界

这个播客对本人的影响还挺大的，我当时还开了会员。很多期的主题即使是放到现在也不过时，但是当时整个播客市场不像现在，有吸引力的播客节目没有那么多，中途我也没有继续听下去了。而且这节目本质上就是李如一的价值观输出，缺少了碰撞，我个人一直都不太喜欢听单口。

然后就到了 2019 年，我重新开始听播客，个人觉得听播客最好的时间是做家务和挤车的时候，播客解放了双手（哦耶！）比起闲聊类的更喜欢听有新内容输出的节目，但是这两种我都会挑自己喜欢的话题。

首先是听播客的渠道，我自己在用的有：

播客软件

podcast：苹果自带的播客软件，偶尔会连不上。

Pocket Casts：大多数时间需要翻 q，优势在于基本上世界各地播客都有，最近有用它在听英文播客。

其他几个都是韩国 app，韩语播客还在探索中。

忽左忽右

忽左忽右还是好评比较多的，每期的精彩程度会根据请的嘉宾水平而波动，71《另一个 1968？盘点风起云涌的 2019 年》我挺喜欢的，还做了笔记，还有刘怡老师来的 34《穿行中东腹地》、48《亲历战争余烬》、49《搭顺风车拜访军阀》和 59《到越南去：富士康小道与工厂大迁徙》。

声东击西

一群驻美记者做的播客，最近有一期《不听这期访谈，你或许就不知道刷屏爆款正能量的问题所在》，从标题就很好地反讽了公众号爆款文章。还有别的节目，比如反潮流俱乐部，听了一期关于 drag queen 的，选材比较特别，

天书广播

历史向播客，可以作为对某段历史的入门，还是比较有趣的，涉及的内容还是比较专业向的。

剩余价值

2019 年更新非常稳定，其实算的上一周双更，有时候会出现和我的观点不一样的时候，听一些不同的观点对个人还是有益处的。

展开讲讲

推荐给同样是电视剧爱好者的听众，比较少见的会讲韩剧的播客，不过感觉设备一般，推荐给朋友，朋友对他们的普通话有一点意见。

小声喧哗

好几个月没有更新了，主要是美国文化的观察，对里面部分观点仍然持反对意见。

以上就是经常听的中文播客，最近开始听国外播客了，不过我听国外播客主要是为了练听力和了解时事新闻，所以订阅了很多的新闻节目。NYT 的 The Daily、BBC 的 Global News Podcast，韩国新闻听 JTBC뉴스룸。

非新闻类播客会听 BBC 出的 The Documentary Podcast，挑感兴趣的主题。

2020 应该会去听更多新的播客，过去的一年播客对于我来说其实是一个渠道，一个听到不同观点的渠道，一个探索新领域的渠道，有些事情如果不是在播客里听到，我自己是不会主动去搜的，可能也就错过了一个新的知识。

但是，播客还是不能代替书籍来学习，它是一种闲聊式的知识输入，过分夸大它的作用也是没必要的，刷微博也可以获取信息。

繁体字阅读困难综合症

发表于 2021-05-12 更新于 2022-06-15 分类于 日常随笔

慢慢消失的繁体字阅读能力。

最近在读《圈外編輯》，图书馆还没采购新版，但是有一本台版书，所以就直接借回来了。结果没看几页实在不能适应竖排，繁体倒还好尚可阅读。

第一章的标题是<做一本书要从何开始？>，作者给出的答案是有手指就能做书。只要有想书写的意愿，有想发表的观点，那就表达出来吧！之前看《东京本屋》，本物（ ほんや ）应该算是书店的另一种表达，其中有提到一家可以寄售zine的书店——模索舍，zine就类似于独立刊物。

我在模索舍看到的是人们的欲望、希望、渴望的原型，也是原石。那些粗糙的外观（有些“书”是作者自己复印并用订书机订的）、不成熟的语言与插图（高中生办的同人志那种）、言论激烈的讨论（极右或极左等等）……模索舍里满是理想、愤怒、人情的暴风雨，难怪这里找不到小清新风格的coffee table book。

其实，比如像笔者在豆瓣上写日记也是一种输出，但和制作出zine还是不一样的。当文字实物化后它就有了重量，装订成册是一种仪式，在物体上打上了时代的烙印。比起统一发行的杂志，zine的灵魂是个人而私密的。

zine算不算非法出版呢？

重要的可能是制作zine的过程，而不是把它卖出去。《圈外编辑》中讲了一个故事，90年代初，作者在德国参加一个书展，遇到了一个俄罗斯人，他的书是用手动打字机和复写纸完成的，只要够用力，一次就能写五张纸，所以他的书就只有五本。

扯远了，继续讲回繁体字，笔者的繁体字阅读能力主要是来源于多年漫画和字幕阅读经验，所以为什么汉化组都喜欢用繁体呢？百度了一下，发现原来很多人都问过这个问题，找到了一个字幕组的回答，看到的是图片版，那总结一下：

1）历史原因

大部分漫画是先有的台版或者港版，那大陆在翻译的时候也会借鉴；早期的汉化组中用繁体字的人比较多。

2）美观原因

早期汉化的漫画以日本漫画为主，众所周知，日语有汉字，使用繁体和原文长得比较像。然后评论下面有人也提到繁体的遮盖面积比较大，不会留下太多的空白，不会给修图人员增加工作负担。

但随着国内版权的发展，购入正版之后制作的翻译字幕多半都是简体，所以现在的年轻一代不习惯阅读繁体了，患上了繁体字阅读困难综合症。生活中很难有其它机会接触繁体字了，即使笔者能阅读繁体字，却不会写，明明也写过几年书法。

虽然不会消亡，但它的存在感正在慢慢减弱。世界上每分每秒都有在消失的东西，见证事物消失很悲哀，有的时候也会想，其实生活永远在继续，时间永远在前进。悲伤的不是消失的现实，而是消失这个过程，似乎听起来有点强说愁的意味。

正因为会想些有的没的，才能感知到自己还活着。
